# Gothic Noir UI Redesign

## Complete UI Transformation with Gothic Noir Color Palette

Your UI has been completely redesigned using the sophisticated Gothic Noir color palette.

## Color Palette Applied

- **Black (#000000)**: Primary backgrounds, headers, text accents
- **Light Gray (#D1D0D0)**: Main content background, text
- **Taupe (#988686)**: Borders, secondary text, subtle accents
- **Dark Taupe (#5C4E4E)**: Hover states, shadows, depth

## Design Changes

### 1. Background
- Dark gradient from pure black to dark taupe
- Sophisticated moody atmosphere
- Professional medical application feel

### 2. Typography
- Lightweight fonts (300-400 weight)
- Uppercase headings with letter-spacing
- Clean, minimal, professional

### 3. Layout
- Sharp corners (border-radius: 0)
- Clean lines and geometric shapes
- No rounded elements for modern architectural feel

### 4. Cards
- Light gray backgrounds
- Strong black borders
- Subtle shadows for depth
- Clean rectangular design

### 5. Header
- Pure black background
- Light gray text
- Taupe border accent
- Minimalist design

### 6. Buttons
- Black background with gray borders
- Uppercase text with letter-spacing
- Fill animation on hover
- Professional and clean

### 7. Form Elements
- Bordered inputs with taupe accents
- Black radio button accents
- Selected state uses darker tones
- Left border indicators

### 8. Results Section
- Black accents throughout
- Taupe borders for separation
- Clean list items with left borders
- Minimal decorations

### 9. Model Banner
- Pure black background
- Light gray text
- Subtle gradient overlay
- Professional presentation

### 10. No Emojis
- All emojis removed from UI
- Clean professional text only
- Medical/clinical aesthetic

## Design Philosophy

- **Sophistication**: Moody, mature color palette
- **Minimalism**: No unnecessary decorations
- **Professionalism**: Medical application standard
- **Clarity**: High contrast for readability
- **Modern**: Sharp geometric design

## Visual Characteristics

- High contrast black/gray scheme
- Sharp rectangular elements
- Minimal animations (only functional)
- Strong borders for definition
- Subtle hover states
- Clean typography
